 ------------------------------------------------------------------

 wrapper subroutine for forward Fourier transform by FFTW3

   a_{k} = \frac{2}{Nfft} \sum_{j=0}^{Nfft-1} x_{j} \cos (\frac{2\pi j k}{Nfft})
   b_{k} = \frac{2}{Nfft} \sum_{j=0}^{Nfft-1} x_{j} \cos (\frac{2\pi j k}{Nfft})

   a_{0} = \frac{1}{Nfft} \sum_{j=0}^{Nfft-1} x_{j}
    K = Nfft/2....
   a_{k} = \frac{1}{Nfft} \sum_{j=0}^{Nfft-1} x_{j} \cos (\frac{2\pi j k}{Nfft})

 ------------------------------------------------------------------

      subroutine FFTW_backward_SMP(plan_backward, Nsmp, Nstacksmp,    &
     &          Ncomp, Nfft, X, X_FFTW, C_FFTW)
 ------------------------------------------------------------------

 wrapper subroutine for backward Fourier transform by FFTW3

   x_{k} = a_{0} + (-1)^{j} a_{Nfft/2} + sum_{k=1}^{Nfft/2-1}
          (a_{k} \cos(2\pijk/Nfft) + b_{k} \sin(2\pijk/Nfft))

 ------------------------------------------------------------------

       i = 1:     a_{0}
       i = 2:     a_{Nfft/2}
       i = 3:     a_{1}
       i = 4:     b_{1}
       ...
       i = 2*k+1: a_{k}
       i = 2*k+2: b_{k}
       ...
       i = Nfft-1:   a_{Nfft/2-1}
       i = Nfft:     b_{Nfft/2-1}

 ------------------------------------------------------------------


Parameters
NsmpNumber of SMP processors
Nstacksmp(0:Nsmp)End number for each SMP process
NcompNumber of components for Fourier transforms
NfftData length for eadh FFT
X(Ncomp,Nfft)Data for Fourier transform


Parameters
plan_forwardFFTW plan for forward transform
plan_backwardFFTW plan for backward transform
aNfftnormalization parameter for FFTW (= 1 / Nfft)
X_FFTWreal data for multiple Fourier transform
C_FFTWspectrum data for multiple Fourier transform
