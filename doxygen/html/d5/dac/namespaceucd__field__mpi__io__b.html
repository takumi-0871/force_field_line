   Data format for the merged binary field data
     1.   Number of process
     2.   Time step
     3.   Time, Delta t
     4.   Stacks of numbe of data points
     5.   Number of fields
     6.   List of number of components
     7.   Field names
     8.   List of data size (Byte)
     9.   All Field data
