    Format of band matrix
               | a(2,1)  a(1,2)  ........     0         0     |
               | a(3,1)  a(2,2)  ........     .         .     |
               |   0     a(3,2)  ........     .         .     |
    a(i,j)  =  |   .       0     ........     0         .     |
               | ...... a(3,k-1)  a(2,k)  a(1,k+1) .......... |
               |   .       .     ........  a(1,N-2)     0     |
               |   .       .     ........  a(2,N-2)  a(1,N-1) |
               |   0       0     ........  a(3,N-2)  a(2,N-1) |

   Original band matrix
      band_a(i-j+iband+1,j) = a(i,j)
      band_a(k,j) = a(k+j-iband-1,j)
   3-band matrix
      band_a(i-j+2,j) = a(i,j)
      band_a(k,j) = a(k+j-2,j)
   5-band matrix
      band_lu(i-j+3,j) = a(i,j)
      band_lu(k,j) = a(k+j-3,j)
   7-band matrix
      band_lu(i-j+4,j) = a(i,j)
      band_lu(k,j) = a(k+j-4,j)


Parameters
nriNumber of radial points
jmaxNumber of spherical harmonics modes
kr_stStart radial address to construct matrix
kr_edEnd radial address to construct matrix
kr_inRadial address for inner boundary
kr_outRadial address for outer boundary
coef_dCoefficient of diffusiotn term
coef_pCoefficient of pressure gradient
