Parameters
n_pointNumber of points for spectrum data
jmaxNumber of modes for local spectrum
kr_bc_stStart radial address to set fixed field
kr_bc_edEnd radial address to set fixed field
fixed_bc(jmax)Boundary condition spectrum
kr_inRadial ID for inner boundary
kr_outRadial ID for outer boundary
r_ICB(0:2)Radius at ICB
r_CMB(0:2)Radius at CMB
fdm2_fix_dr_ICB(-1:1,3)Matrix to evaluate field at ICB with fixed radial derivative
fdm2_fix_dr_CMB(-1:1,3)Matrix to evaluate field at CMB with fixed radial derivative
flux_ICB(jamx)Spectrum of fixed flux at ICB
flux_CMB(jamx)Spectrum of fixed flux at CMB
coef_impCoefficient for contribution of implicit term
coef_dCoefficient for magnetic diffusion
is_fldInput field address for d_rj
ntot_phys_rjTotal number of components
d_rjSpectrum data
