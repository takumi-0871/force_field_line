NBNumber of components for communication
NPNumber of node
NNumber of internal node

NEIBPETOTNumber of processses to send
NEIBPE(NEIBPETOT)Process ID to communicate
STACK_EXPORT(0:NEIBPETOT)End points of send buffer for each process
NOD_EXPORT(STACK_EXPORT(NEIBPETOT))local node ID to copy in send buffer

STACK_IMPORT(0:NEIBPETOT)End points of receive buffer for each process
NOD_IMPORT(STACK_IMPORT(NEIBPETOT))local node ID to copy from receive buffer
REV_IMPORT(NP-N)import buffer ID for each data point

X(NB*NP)vector to syncronize

X(6*NP)Six components of received data

X(3*NP)Three components of received data

X(2*NP)Two components of received data

X(NP)Scalar received data

iX(NP)Integer received data
