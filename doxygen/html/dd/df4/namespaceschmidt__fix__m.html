* Legendre and adjoint Legendle Polynomial for order m
*
* p(l) : Schmidt Polynomial
* Normalization...
* P_{l}^{0} = P_{l,0}
* P_{l}^{m} = sqrt( 2(l-m)! / (l+m)! ) * P_{l,m}
* theta : input colatitude ( 0 =< theta <= pi )
*
* df(l) : work area
